# Creating PDFs Programmatically

Three approaches depending on complexity:
- **fpdf2** — lightweight pure-Python, no dependencies, great for simple reports
- **WeasyPrint** — HTML+CSS → PDF, perfect for styled documents designed in HTML
- **pypdf** — merge, split, watermark existing PDFs

## fpdf2 — Lightweight PDF Generation

```python
# install_package("fpdf2")
from fpdf import FPDF, XPos, YPos

class PDF(FPDF):
    def header(self):
        self.set_font("Helvetica", "B", 15)
        self.cell(0, 10, "MagAgent Report", new_x=XPos.LMARGIN, new_y=YPos.NEXT, align="C")
        self.ln(5)

    def footer(self):
        self.set_y(-15)
        self.set_font("Helvetica", "I", 8)
        self.cell(0, 10, f"Page {self.page_no()}/{{nb}}", align="C")

pdf = PDF()
pdf.alias_nb_pages()
pdf.add_page()

# Title
pdf.set_font("Helvetica", "B", 16)
pdf.cell(0, 10, "Executive Summary", new_x=XPos.LMARGIN, new_y=YPos.NEXT)
pdf.ln(5)

# Body text
pdf.set_font("Helvetica", size=12)
pdf.multi_cell(0, 8, "This report was generated automatically by MagAgent. "
               "It summarizes the key findings from the analysis.")
pdf.ln(10)

# Table
pdf.set_font("Helvetica", "B", 11)
col_widths = [60, 40, 40, 40]
headers = ["Name", "Dept", "Score", "Grade"]
for header, w in zip(headers, col_widths):
    pdf.cell(w, 8, header, border=1, align="C")
pdf.ln()

pdf.set_font("Helvetica", size=10)
data = [("Alice Smith", "Eng", "95", "A"), ("Bob Jones", "QA", "82", "B")]
for row in data:
    for val, w in zip(row, col_widths):
        pdf.cell(w, 7, val, border=1)
    pdf.ln()

# Image
pdf.ln(10)
pdf.image("chart.png", x=30, w=150)   # if chart.png exists

pdf.output("report.pdf")
print("Saved report.pdf")
```

## WeasyPrint — HTML + CSS → PDF (Best for Styled Docs)

```python
# install_package("weasyprint")
from weasyprint import HTML, CSS

html_content = """
<!DOCTYPE html>
<html>
<head>
<style>
  body { font-family: Arial, sans-serif; margin: 40px; }
  h1 { color: #2563EB; border-bottom: 2px solid #2563EB; }
  table { width: 100%; border-collapse: collapse; }
  th { background: #2563EB; color: white; padding: 8px; }
  td { border: 1px solid #ddd; padding: 8px; }
  tr:nth-child(even) { background: #f0f4ff; }
  @page { margin: 2cm; }
</style>
</head>
<body>
  <h1>Project Report</h1>
  <p>Generated by MagAgent on 2026-06-05</p>
  <table>
    <tr><th>Name</th><th>Status</th><th>Score</th></tr>
    <tr><td>Alice</td><td>Active</td><td>95</td></tr>
    <tr><td>Bob</td><td>Active</td><td>88</td></tr>
  </table>
</body>
</html>
"""

HTML(string=html_content).write_pdf("styled_report.pdf")
```

## pypdf — Merge / Split / Watermark

```python
# install_package("pypdf")
from pypdf import PdfReader, PdfWriter

# Merge two PDFs
writer = PdfWriter()
for filename in ["part1.pdf", "part2.pdf"]:
    reader = PdfReader(filename)
    for page in reader.pages:
        writer.add_page(page)
with open("merged.pdf", "wb") as f:
    writer.write(f)

# Extract pages 1-3
reader = PdfReader("big_doc.pdf")
writer = PdfWriter()
for page in reader.pages[:3]:
    writer.add_page(page)
with open("excerpt.pdf", "wb") as f:
    writer.write(f)
```

## Decision Guide
| Need | Use |
|---|---|
| Simple text + tables | `fpdf2` |
| Full HTML/CSS design | `WeasyPrint` |
| Merge/split existing PDFs | `pypdf` |
| Complex pixel-perfect layout | `reportlab` |
